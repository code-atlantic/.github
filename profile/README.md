## Open Source

We maintain a growing collection of open-source libraries and tools for the WordPress ecosystem and beyond:

- **[Chophper](https://github.com/code-atlantic/chophper)** — PHP utility for HTML-aware text truncation that preserves tag structure. Truncate by characters, words, sentences, or paragraphs.
